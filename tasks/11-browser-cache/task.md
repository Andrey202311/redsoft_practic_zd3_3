# 11. Кэш браузера

## Задание
При первой загрузке или жестком обновлении страницы картинка с котиком загружается с сервера, а при обычном обновлении страницы — из кэша браузера.

## Проверка
Открыта вкладка Network в DevTools. Выполнено обычное обновление страницы (F5).

## Фактический результат
При обычном обновлении картинка cat.png загружается из кэша (столбец "Размер" показывает "из кэша", время загрузки 0 мс).

## Ожидаемый результат
Картинка загружается из кэша при обычном обновлении.

## Вывод
Информация о кэшировании найдена. Кэширование работает корректно.
