# 03. JSON в запросе

## Задание
При нажатии на кнопку отправляется запрос на бэкенд. JSON в боди запроса имеет вид:
```json
{"name": "Андрей"}
```

## Проверка
Открыта вкладка Network в DevTools, выбран запрос, перейдено на вкладку "Запрос" (Payload).

## Фактический результат
Боди запроса пустое.

## Ожидаемый результат
В теле запроса присутствует JSON с полем name.

## Вывод
Обнаружен дефект: JSON не передается в теле запроса.
