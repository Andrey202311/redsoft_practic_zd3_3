# 01. Метод запроса

## Задание
При нажатии на кнопку отправляется POST запрос на бэкенд.

## Проверка
Открыта вкладка Network в DevTools, нажата кнопка "Отправить". Просмотрен столбец "Метод".

## Фактический результат
Метод запроса: PUT

## Ожидаемый результат
Метод запроса: POST

## Вывод
Обнаружен дефект: запрос отправляется методом PUT вместо ожидаемого POST.
