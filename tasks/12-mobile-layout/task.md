# 12. Мобильная верстка

## Задание
При вертикальной ориентации мобильная верстка страницы соответствует макету в Figma.

## Проверка
Включен Toggle Device Toolbar (Ctrl+Shift+M), выбрано мобильное устройство, сравнена верстка с макетом Figma.

## Фактический результат
Мобильная верстка соответствует макету из Figma.

## Ожидаемый результат
Соответствие макету Figma при вертикальной ориентации.

## Вывод
Работает корректно. Мобильная верстка соответствует макету.
