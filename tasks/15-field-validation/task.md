# 15. Валидация поля

## Задание
При отправке формы происходит проверка, что поле "Email" соответствует формату адреса электронной почты (наличие "@" и др.). Проверка происходит на клиенте и бэкенде.

## Проверка
1. В браузере: введен корректный email test@example.com — форма отправлена успешно.
2. В Postman: отправлен POST-запрос на https://devtools.qa.studio/tasks/field_validation с телом `{"email":"test"}` (без символа @).

## Фактический результат (Postman)
- Статус ответа: 200 OK
- Тело ответа:
```json
{
  "message": "Адрес электронной почты успешно получен",
  "data": {
    "email": "test"
  }
}
```

## Ожидаемый результат
Статус 400 Bad Request или сообщение об ошибке валидации.

## Вывод
Обнаружен дефект: валидация email на бэкенде не реализована. Сервер принимает строку без символа "@" как валидный email.
