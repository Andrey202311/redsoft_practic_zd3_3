# 08. Query параметры

## Задание
При нажатии на кнопку отправляется запрос на бэкенд. В query передается параметр section со значением 3.

## Проверка
Открыта вкладка Network в DevTools, просмотрен URL запроса.

## Фактический результат
URL запроса содержит `?section=3`.

## Ожидаемый результат
В query-строке присутствует параметр section=3.

## Вывод
Работает корректно. Query-параметры передаются в запросе.
