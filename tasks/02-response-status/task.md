# 02. Статус ответа

## Задание
При нажатии на кнопку отправляется запрос на бэкенд. Ответ должен иметь статус 201.

## Проверка
Открыта вкладка Network в DevTools, нажата кнопка "Отправить". Просмотрен столбец "Статус".

## Фактический результат
Статус ответа: 201

## Ожидаемый результат
Статус ответа: 201

## Вывод
Работает корректно. Статус ответа соответствует требованиям.
