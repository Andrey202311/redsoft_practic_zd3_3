# 09. Local storage

## Задание
При нажатии на кнопку значение поля "Имя" сохраняется в local storage браузера с ключом name.

## Проверка
Открыта вкладка Application в DevTools, раздел Local Storage, выбран домен devtools.qa.studio.

## Фактический результат
Ключ name со значением "Андрей" присутствует в Local Storage.

## Ожидаемый результат
Значение сохранено с ключом name.

## Вывод
Работает корректно. Значение корректно сохранено в Local Storage.
