# 10. Session storage

## Задание
При нажатии на кнопку значение поля "Имя" сохраняется в session storage браузера с ключом name.

## Проверка
Открыта вкладка Application в DevTools, раздел Session Storage, выбран домен devtools.qa.studio.

## Фактический результат
Ключ name со значением "Андрей" присутствует в Session Storage.

## Ожидаемый результат
Значение сохранено с ключом name.

## Вывод
Работает корректно. Значение корректно сохранено в Session Storage.
