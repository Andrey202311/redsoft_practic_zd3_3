# 06. Заголовки запроса

## Задание
При нажатии на кнопку отправляется запрос на бэкенд. Запрос содержит заголовок X-User-Id.

## Проверка
Открыта вкладка Network в DevTools, выбран запрос, перейдено на вкладку "Заголовки", просмотрен раздел "Заголовки запроса".

## Фактический результат
Заголовок X-User-Id присутствует в запросе со значением UUID.

## Ожидаемый результат
Заголовок X-User-Id передается в запросе.

## Вывод
Работает корректно. Заголовок передается в запросе.
