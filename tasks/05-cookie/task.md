# 05. Cookie

## Задание
При нажатии на кнопку отправляется POST запрос на бэкенд. В хедере ответа set-cookie придет кука devtools_session_id со свойствами HttpOnly, Secure и сроком жизни 1 год (365 дней).

## Проверка
Открыта вкладка Network в DevTools, выбран запрос, перейдено на вкладку "Заголовки", просмотрен раздел "Заголовки ответа".

## Фактический результат
Кука devtools_session_id установлена с параметрами:
- HttpOnly
- Secure
- Max-Age=31536000 (365 дней)

## Ожидаемый результат
Кука с именем devtools_session_id, флагами HttpOnly и Secure, сроком 365 дней.

## Вывод
Работает корректно. Cookie установлена с требуемыми параметрами.
