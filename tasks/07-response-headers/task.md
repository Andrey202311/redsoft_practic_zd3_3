# 07. Заголовки ответа

## Задание
При нажатии на кнопку отправляется запрос на бэкенд. Ответ содержит заголовок X-Request-Id.

## Проверка
Открыта вкладка Network в DevTools, выбран запрос, перейдено на вкладку "Заголовки", просмотрен раздел "Заголовки ответа".

## Фактический результат
Заголовок X-Request-Id присутствует в ответе сервера.

## Ожидаемый результат
Заголовок X-Request-Id передается в ответе сервера.

## Вывод
Работает корректно. Заголовок передается в ответе.
