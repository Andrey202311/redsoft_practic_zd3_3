# 04. JSON в ответе

## Задание
При нажатии на кнопку отправляется запрос на бэкенд. В боди ответа должны получить JSON:
```json
{"qa_studio_student": false}
```

## Проверка
Открыта вкладка Network в DevTools, выбран запрос, перейдено на вкладку "Ответ".

## Фактический результат
Сервер прислал корректный JSON: {"qa_studio_student": false}

## Ожидаемый результат
JSON с полем qa_studio_student со значением false.

## Вывод
Работает корректно. Сервер возвращает ожидаемый JSON.
